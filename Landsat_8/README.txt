The file "Landsat8_notebook.ipynb" contains code used for downloading products from USGS M2M Api, pre-process the acquired data and fuse them in a NETCDF file.

The file "main_landsat8_Download.py" contains code used for downloading products from USGS M2M Api only.

Both codes uses functions contained inside the "Functions" folder.

Both codes need a .txt file with the token necessary to access the USGS M2M api
